# Club Animo Webサイト コンテンツ制作 / 撮影ディレクション設計書

0 目的
本設計書はClub Animo Webサイトで使用するコンテンツ制作ルール、撮影ディレクション、文章制作基準を定義する。写真とコピーの品質がサイト価値を大きく左右するため、統一ルールを設ける。

---

1 コンテンツ構成 / 2 コンセプト

- 構成: 店舗紹介 / キャスト紹介 / 店内ギャラリー / イベント情報 / 求人情報 / ニュース
- コンセプト: 明るい高級クラブ（大きなシャンデリア / 広い店内 / 富裕層向け）

---

3 写真の方向性

- トーン: 明るい / 高級感 / 洗練 / モダン

---

4 店内撮影

- 対象: シャンデリア / メインフロア / VIP席 / バーカウンター
- ポイント: 広さ / 豪華さ / 照明

---

5 キャスト撮影（顔出し不可対応）

- アングル: 後ろ姿 / 横顔 / シルエット / 手元
- 雰囲気: ドレス / 店内背景 / 高級感

---

6 キャスト紹介文

- 構成: キャッチコピー / プロフィール / 趣味 / 一言
- 例: 「落ち着いた大人の魅力」「お酒が好きです」「一緒に楽しい時間を過ごしましょう」

---

7 店舗紹介コピー / 8 ギャラリー構成

- トップページコピー例: 「関内の大人の社交場」「煌びやかなシャンデリアの下で 特別な時間を」
- ギャラリー: 店内 / シャンデリア / VIP席 / カウンター / イベント

---

9 イベントコンテンツ / 10 求人コンテンツ

- イベント: 周年イベント / シャンパンフェア / 特別イベント
- キャスト求人: 働きやすさ / 高収入 / 未経験歓迎
- スタッフ求人: 店舗運営 / 接客 / マネジメント

---

11 求人写真 / 12 コピーライティング / 13 NG表現

- 写真: 働くイメージ / 店内雰囲気 / ドレス姿
- コピー: 短く / 高級感 / 読みやすい
- NG表現: 過度な誇張 / 違法表現 / 誤解を招く表現

---

14 SEOコンテンツ

- 以下を自然に入れる: キャバクラ / 高級クラブ / 関内 / 横浜

---

15 画像品質

- 最低解像度: 1200px以上 / 画質劣化禁止

---

16 SNS連携 / 17 更新頻度 / 18 コンテンツ管理

- SNS: Instagram投稿 / イベント告知
- 更新頻度: イベント(月1回) / ニュース(月1回)
- 管理: CMSからキャスト、イベント、ニュース等の更新を可能にする

---

19 将来拡張 / 20 受け入れ基準

- 将来拡張: 動画コンテンツ / 店内紹介動画
- 受け入れ: 高級感のある写真 / 統一されたコピー / キャスト魅力訴求 / 求人訴求

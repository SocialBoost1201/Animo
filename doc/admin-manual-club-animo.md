# CLUB Animo 管理画面（CMS）操作マニュアル

本マニュアルは、CLUB Animo の店舗スタッフ様が、日々のWebサイト運用（キャスト情報更新・シフト管理・お知らせ更新など）を行うための手順書です。スマートフォン・タブレット・PCのどの端末からでも操作可能です。

---

## 1. 管理画面へのログイン

管理画面はセキュリティ保護のため、許可されたスタッフ様のみがアクセスできます。

1. ブラウザで **`https://[本番ドメイン]/admin`** にアクセスします。
2. ログイン画面が表示されます。
3. 店舗責任者より共有された **「メールアドレス」と「パスワード」** を入力し、「ログイン」ボタンを押します。
4. 認証に成功すると、ダッシュボード（本日の出勤人数や新着問い合わせ件数が確認できる画面）が表示されます。

> **※注意**：パスワードを忘れた場合や、退職者のアカウントを無効化したい場合は、システム管理者（またはオーナー）へご連絡ください。

---

## 2. 日々の業務：キャスト出勤（シフト）の登録

毎日の営業前に、本日出勤するキャストをサイト上に表示させるための手順です。

1. 左側メニュー（スマホの場合は右上の「≡」メニュー）から **「シフト管理 (Shifts)」** を選択します。
2. **本日の日付** が選択されていることを確認します（※翌日のシフトを事前登録する場合はカレンダーで日付を変更します）。
3. 画面右上の **「+ 出勤登録」** または **「出勤キャストの追加」** ボタンを押します。
4. 登録済みのキャスト一覧が表示されます。出勤予定のキャスト名横の **チェックボックスをON** にします。
   - ※遅出などがある場合は、備考欄に「21時〜出勤」などと入力します。
5. **「保存する (Save)」** ボタンを押し、「設定しました」の通知が出れば完了です。
   - ※これだけで、サイト上のお客様が見る「本日の出勤一覧」ページに自動で表示されます。

---

## 3. キャスト一覧の追加とプロフィール更新

新しいキャストが入店した際や、写真を変更する際の手順です。

### 3-1. 新規キャストの登録

1. メニューから **「キャスト管理 (Casts)」** を選択します。
2. 画面右上の **「+ 新規追加」** ボタンを押します。
3. 以下の基本情報を入力します。
   - **名前（源氏名）**: （必須）
   - **英語表記**: （任意、URLなどで使われる英字名）
   - **役職**: レギュラー、体験入店など。
4. **「顔写真（プロフィール画像）」**枠の「画像を選択」ボタンを押し、スマホやPC内の写真をアップロードします。
   - ※写真は縦長の長方形（4:5比率）が最も美しく表示されます。
   - ※写真がない場合は空欄のままでも、システムが「高級感のあるシルエット画像」を代理で表示しますのでご安心ください。
5. 入力が完了したら、最下部の **「登録する」** ボタンを押します。

### 3-2. 既存キャストの修正・退店処理

1. メニューから「キャスト管理」を開きます。
2. 一覧から、修正したいキャストの **「編集（ペンマーク）」または「名前部分」** を押します。
3. プロフィールや写真を変更し、「更新する」を押します。
4. **【退店した場合】**: 情報を削除するのではなく、編集画面で「ステータス（Status）」を **公開（Published）から「非公開（DraftまたはArchived）」に変更** して保存します。これによりサイト上からは全く見えなくなりますが、過去の管理データとして残すことができます。

---

## 4. ギャラリー・イベント情報の追加

### 4-1. ギャラリー写真の追加（店内風景など）

1. メニューから **「ギャラリー管理 (Gallery)」** または「コンテンツ管理」内のギャラリーを選択します。
2. **「+ 写真を追加」** を押し、画像をアップロードします。
3. 写真のタイトル（例: 「VIPルームA」等）を入力し、保存します。

### 4-2. 新着お知らせ（News）やイベント情報の追加

1. メニューから **「お知らせ・イベント (News/Events)」** を選択します。
2. **「+ 新規追加」** を押します。
3. **タイトル** （例: 「ハロウィンイベント開催のお知らせ」）、**日付**、**内容** を入力します。
4. 関連するバナー画像があればアップロードし、「保存する」を押します。

---

## 5. お客様からのお問い合わせ・求人応募の確認

サイトの応募フォームから連絡が来た場合の対応手順です。

1. メニューに赤い通知バッジ「1」などが表示されている場合、未読の連絡が届いています。
2. メニューから **「お問い合わせ (Inquiries)」** または **「求人応募 (Applications)」** を開きます。
3. 一覧から未読のメッセージを開くと、お客様の「お名前、連絡先（電話やLINE ID）、問い合わせ内容等」を確認できます。
4. スマホ等から実際にお客様や求職者へ連絡（電話・LINEなど）を行った後、管理画面上でステータスを **「未対応」から「連絡済（または既読・面接予定）」に変更** します。
   - ※これにより、スタッフ間で「誰かが対応済みであること」が共有され、連絡漏れや重複対応を防ぐことができます。

## 6. お困りのときは（トラブルシューティング）

- **Q. 画像がアップロードできない / エラーが表示される**
  - 👉 A: 画像の容量が大きすぎる可能性があります（通常5MB以上はエラーになるか重くなります）。スクリーンショットを撮り直すか、サイズを縮小してから再度お試しください。

- **Q. 「権限がありません」「アクセスが拒否されました」と出る**
  - 👉 A: 一定時間操作をしなかったためログインの期限が切れた可能性があります。一度画面を閉じて、再度 `https://.../admin` にアクセスしログインし直してください。

- **Q. 反映までにラグがある**
  - 👉 A: サイトの表示を極限まで速くするため、お客様に表示される画面は数分間キャッシュ（一時保存）されることがあります。5分ほど待ってからページを再読込してください。

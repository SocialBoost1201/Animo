# Club Animo Webサイト キャストシステム詳細設計書

**ドキュメントID:** detail-design-02-cast-system  
**受領日時:** 2026-03-06

---

## 0. 目的

キャバクラサイトにおいて最も閲覧される「本日の出勤」「所属キャスト一覧」をユーザーが直感的に閲覧できるUIを構築する。

---

## 1. 本日の出勤表示（トップページ）

### 1.1 トップページ表示

- トップページに `Today Cast Section` を配置
- **表示件数:** Desktop: 8〜12件 / Mobile: 横スクロール

### 1.2 表示要素（CastCard）

- image (placeholder) / name / tag / today badge

### 1.3 表示条件と順序

- **条件:** `Cast.isToday = true`
- **優先順位:** 1. 新人 → 2. 人気 → 3. 通常

---

## 2. 本日の出勤専用ページ（URL: `/shift`）

### 2.1 UI

- Today Tab / Week Tab

### 2.2 Today表示

- 今日の出勤キャストをGrid表示

---

## 3. 所属キャスト一覧（URL: `/cast`）

### 3.1 一覧構造（CastGrid）

- **カラム構成:** Desktop: 4列 / Tablet: 3列 / Mobile: 2列

### 3.2 フィルタ機能

- **雰囲気:** 上品 / 華やか / 落ち着き / 会話好き
- **来店目的:** 接待 / 同伴 / 盛り上げ
- **新人**

### 3.3 表示順（デフォルト）

- 新人 → 人気 → 入店順

---

## 4. キャスト詳細ページ（URL: `/cast/[slug]`）

### 4.1 表示要素

- Hero image (placeholder)
- プロフィール（趣味 / 接客タイプ）
- 出勤予定
- 予約ボタン

---

## 5. 管理画面仕様

### 5.1 Cast管理（CRUD）

- **プロパティ:** name, slug, tags, intro, images, isToday, joinedAt

### 5.2 出勤管理（ShiftEditor）

- 日付プルダウン/カレンダー ＋ キャストチェックボックスのUI

---

## 6. 画像仕様（顔出し不可対応）

- 画像タイプ: 後ろ姿 / 横顔 / ドレス / 雰囲気

---

## 7. アニメーション

- **キャストカード:** hover scale
- **grid load:** stagger animation（スタッガー表示）

---

## 8. SEO

- **キャスト個別URL:** `/cast/[slug]`（キャスト名検索対策）

---

## 9. 受け入れ基準

- トップに本日の出勤が表示される
- `/cast` で所属キャスト一覧が見れる
- `/cast/[slug]` で個別ページがある
- 管理画面から出勤更新ができる

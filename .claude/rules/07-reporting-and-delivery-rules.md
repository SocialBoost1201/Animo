# 07 — Reporting and Delivery Rules

> Repo: Animo | Applies to: Claude Code, Codex, Antigravity

## Completion Report Format

Every task completion must include:
1. Files modified (full paths)
2. Actual diff (before/after or patch)
3. Verification commands run and output
4. Scope compliance confirmation
5. Skill Usage Report (used / not used / why)

Summary-only completion reports are invalid.

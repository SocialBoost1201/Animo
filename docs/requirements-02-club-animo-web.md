# Club Animo Webサイト 要件定義書（改訂版）

**ドキュメントID:** requirements-02  
**受領日時:** 2026-03-06

---

## 0. 目的

Antigravityが参考サイトをURL解析し、Club Animoの白基調ハイエンド、最先端モダンサイトを設計・実装するための要件定義。運用を見据え、予約・問い合わせ・求人応募の成果最大化を目的とする。

---

## 1. 前提と制約

### 1.1 ブランドと内装 / 1.2 顧客 / 1.3 キャスト掲載 / 1.4 料金公開

- 明るい内装・シャンデリア特化・富裕層ターゲット・顔出し不可の制約・ボトル価格非公開（※01から変更なし）

### 1.5 アニメーション方針（NEW）

- 最先端サイトとして、アニメーションは体験を設計する要素としてしっかり使う
- 過剰な点滅、過度なパララックス、重い演出による表示速度低下は禁止

### 1.6 画像差し替え運用（NEW）

- 画像部分は管理者による差し替えを前提とした空ブロック（プレースホルダ）を配置
- 画像未確定でもレイアウトが崩れない設計

### 1.7 運用制約（UPDATE）

- 店長や運営による日々更新（本日出勤など）
- 【追加】スタッフ不足に対応するため、求人の応募導線と内容を強化

---

## 2. 参考サイト分析観点（追加事項）

- スクロール連動演出の粒度とテンポ
- キャスト一覧と詳細ページの情報設計、回遊導線
- 表示速度と読み込み戦略の推定
- **分析アウトプット追加:** Animo用 アニメーション設計方針 / 画像プレースホルダ運用方針

---

## 3. ゴールとKPIの追加

- **追加ゴール:** キャスト求人、スタッフ求人の応募を最大化する
- **追加KPI:** 求人ページ到達率 / 求人応募開始率（フォーム到達） / 求人応募完了率

## 4. サイトマップ（追加）

- Recruit Staff（スタッフ求人）

## 5. コアUIとアニメーション・プレースホルダ要件

### 5.1 共通UI

- 白背景主役、差し色（黒・金）最小限、写真と動画の比重を高くする

### 5.2 アニメーション必須要件（NEW）

- 一貫したモーションルール
- トップはシャンデリア中心にシネマティック演出
- セクション遷移：スクロール連動（フェード・スライド・ズーム・テキストリビール）
- ホバー/タップの気持ち良い反応
- キャスト一覧：カード出現演出・フィルタ並び替え演出
- 料金ページ：段階的な表示演出
- 速度は上品・遅め。動きは小さく回数で体験を作る
- `motion-reduced` 環境での演出抑制対応

### 5.3 画像プレースホルダ必須要件（NEW）

- 画像未設定時でも余白とレイアウトが崩れない空ブロック
- 必須機能: 固定アスペクト比 / ローディング骨組み表示 / 代替テキスト枠 / 管理画面紐付けID
- トップ・ギャラリー・キャスト・求人で共通の仕組みを利用

## 7. 求人要件：キャスト求人ページ（NEW / 必須）

### 7.1 目的

- 顔出し制約下でも応募増 / 店の格と安全性を伝え安心感を担保

### 7.2 構成（必須セクション）

- FV応募CTA常設 / 働く価値（シャンデリア・客層） / 1日の流れ / 応募条件 / 待遇（時給・バック・罰金有無・送り・衣装等） / 安心安全（反社排除・飲酒強要禁止等） / FAQ

### 7.3 フォーム項目

- 氏名（仮名可） / 年齢 / 連絡先 / 希望勤務日数・時間帯 / 経験有無 / 面談希望日 / ひとこと

### 7.4 アニメーション要件

- 段階的出現 / 待遇比較表の行リビール / 応募CTAのスクロール連動出し分け

---

## 8. 求人要件：スタッフ求人ページ（NEW / 必須・強化）

### 8.1 目的

- スタッフ不足解消のため応募導線を最重要視。業務と待遇、成長メリットを明確化

### 8.2 募集職種

- ホール / ボーイ / 店長候補 / ドライバー（必要時）

### 8.3 構成（必須セクション）

- FV応募CTA常設 / スタッフ不足の現状と改善意志の宣言 / 職種別仕事内容 / 1日の流れ / 求める人物像 / 待遇（固定給・インセンティブ・昇給・まかない等） / 成長（店長候補ルート等） / 職場環境（クリーンな運営） / FAQ

### 8.4 フォーム項目

- 氏名 / 年齢 / 連絡先 / 希望職種 / 希望シフト / 経験有無 / 面談希望日 / ひとこと

### 8.5 アニメーション要件

- 職種カードホバー展開 / 仕事内容・待遇のストーリー進行スクロール連動 / 応募ステップのプログレス表示

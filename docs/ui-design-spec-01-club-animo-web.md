# Club Animo Webサイト UIデザイン設計書

**ドキュメントID:** ui-design-spec-01  
**受領日時:** 2026-03-06

---

## 0. 目的

UIデザインルール、ビジュアルガイドライン、コンポーネントスタイル、レイアウト基準、タイポグラフィ、カラーパレット、画像配置ルールを定義する。白基調で高級感があり、最先端モダンなUIを構築する。

**実現する価値:**

- 高級キャバクラとしての格 / 明るい内装の魅力を最大化 / シャンデリアを象徴としたブランド体験
- 顔出し不可キャストでも成立するUI / 求人応募を増やす導線

---

## 1. デザインコンセプト

- **キーワード:** Luxury / Modern / Minimal / Elegant / Bright
- **イメージ:** 高級 / 明るい / 洗練 / 上品（暗いラウンジではなく明るい高級クラブ）

---

## 2. カラーパレット

- **Base:** White (`#FFFFFF`) - 背景の主役
- **Text:** Primary (`#111111`) / Secondary (`#555555`)
- **Accent:** Gold (`#C9A86A`) - CTA、ライン、アクセントに使用
- **Auxiliary:** Light Gray (`#F5F5F5`) - 背景ブロック

---

## 3. タイポグラフィ

- **英字:** Serif系（高級感演出）
- **日本語:** 読みやすいゴシック
- **見出しサイズ:** H1(48px) / H2(36px) / H3(28px) / Body(16px)

---

## 4. レイアウトグリッド

- **Desktop:** 1200px container / 12 column grid
- **Tablet:** 768px container
- **Mobile:** Full width

---

## 5. 余白設計

- **セクション間:** 120px
- **要素間:** 32px
- **カード内:** 16px

---

## 6. 画像設計（プレースホルダ比率）

※すべての画像はプレースホルダブロックを使用

- **Hero / Recruit:** 16:9
- **Cast:** 4:5
- **Gallery:** 3:2

---

## 7. ボタンUI

- **Primary:** 背景 Gold / 文字 White
- **Secondary:** 枠線 Black / 背景 Transparent
- **Hover:** 軽く浮く

---

## 8. カードUI

- **Cast Card:** 白背景 / 薄い影
- **News Card:** シンプル
- **Recruit Card:** 強調

---

## 9. フォームUI

- **入力欄:** 角丸
- **Focus:** 枠色変更
- **Error:** 赤テキスト

---

## 10〜11. ヘッダー・フッター

- **Header:** Logo / Menu / CTA （Sticky Headerはスクロールで縮小）
- **Footer:** Access / SNS / Recruit / Policy

---

## 12. キャストUI（顔出し不可前提）

- **写真:** 後ろ姿
- **タグ表示:** 上品 / 華やか / 落ち着き

---

## 13. 求人UI

- **Cast Recruit:** 女性向けデザイン
- **Staff Recruit:** 男性向けデザイン

---

## 14. Night Style診断

- カードUI / 選択ボタン / 結果カードのスタイル適用

---

## 15〜18. モバイル・SEO・その他

- **モバイル:** キャスト横スクロール / 固定CTA
- **アクセス:** Map / 住所コピー
- **SEO:** パンくず / 構造化データ
- **管理UI:** シンプル

---

## 19. 受け入れ基準

- 白基調で高級感がある
- キャスト顔出し無しでも成立する
- 求人導線が強い
- モーションと調和している
- スマホで操作しやすい

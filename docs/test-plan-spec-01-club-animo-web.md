# Club Animo Webサイト テスト設計書

0 目的
本設計書はClub Animo Webサイトの品質保証を目的としたテスト計画を定義する。
要件（高級感のあるUI表示、キャスト関連表示、イベント・求人、管理画面操作、モバイル最適化、SEO）を満たしているかを検証する。

---

1 テスト範囲

- 対象ページ: トップ/キャスト一覧/詳細/本日の出勤/料金/ギャラリー/イベント/ニュース/求人/応募フォーム/問い合わせフォーム/管理画面

---

2 テスト環境

- ブラウザ: Chrome / Safari / Edge / Firefox
- OS: MacOS / Windows / iOS / Android
- 画面サイズ: Desktop / Tablet / Mobile

---

3 UIテスト

- デザイン確認: 白基調の高級感 / フォント表示 / 余白 / ボタン配置 / レスポンシブ表示

---

4 ナビゲーションテスト

- メニューリンク / 全ページ遷移 / パンくず / フッターリンク

---

5 キャスト機能テスト

- 一覧表示 / カード表示 / 詳細ページ / タグ表示 / 画像表示

---

6 本日の出勤テスト

- 出勤キャスト表示 / 日付切替 / 出勤時間表示

---

7 ギャラリーテスト

- 店内写真表示 / 拡大表示 / モバイル対応

---

8 イベント/ニュース機能テスト

- 一覧表示 / 詳細表示 / 画像表示

---

10 求人ページ/問い合わせテスト

- キャスト・スタッフ求人ページ表示
- 応募/問い合わせフォーム機能（送信確認 / DB保存確認）

---

12 管理画面テスト

- ログイン / キャスト登録 / 出勤更新 / イベント登録 / ニュース登録 / 画像アップロード

---

13 アニメーションテスト

- 画面崩れ / タップ操作 / スクロール

---

18 アクセシビリティテスト

- altテキスト / キーボード操作 / コントラスト

19 ユーザーテスト

20 受け入れ基準

- 全ページ正常表示 / フォーム送信成功 / 管理画面操作可能 / モバイル正常表示 / SEO設定完了

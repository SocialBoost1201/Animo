**作成日:** 2026-03-06

## 1. 各サイト概要

### 1-A. XEE（六本木ニュークラブ）

**URL:** https://www.club-xee.com/

| 項目        | 内容                                                           |
| ----------- | -------------------------------------------------------------- |
| 業態        | ニュークラブ（六本木 2F/3F）                                   |
| ナビ        | 会員登録・マイページあり / Instagram・TikTok連携               |
| 特徴        | ランキング・ブログ・ニュース更新頻度高 / ファンクラ外部CMS連携 |
| キャスト    | 写真付きカード / 個別ブログ / 誕生日カレンダー                 |
| 更新運用    | ファンクラ依存・独自CMSは限定的                                |
| CTAパターン | 電話番号常設 / 会員登録ボタン                                  |

**良い点**

- キャスト個別ブログで毎日コンテンツ更新 → SEO・リピーター醸成
- ランキング機能でエンゲージメント向上
- TikTok/Instagram連携で若年層への接触

**弱い点**

- 高級感に乏しい（ポータルサイト的UIで汎用感が強い）
- ファンクラ依存のため独自世界観の確立が困難
- 予約導線が不明確（電話のみ推定）
- ブランドより更新量優先の設計

- 料金の具体的な数字が不透明

- 出勤表・本日出勤情報なし（即時来店トリガーがない）
- スマホ最適化が不十分な可能性

## 2. 競合共通点

| 要素               | 共通パターン                                                             |
| ------------------ | ------------------------------------------------------------------------ |
| キャスト掲載       | 顔出し中心 / 個別ページあり                                              |
| 料金               | 具体数字の提示は控えめ                                                   |
| 出勤表             | ブログで代替、専用出勤表は少ない                                         |
| SNS                | Instagram常設 / TikTokも増加傾向                                         |
| 予約               | 電話中心 / フォームは少数                                                |
| ナビ               | キャスト・料金・アクセスが必須3項目                                      |
| **アニメーション** | スクロール連動は基本レベル・シネマティックな深い没入感を持つサイトはない |
| **画像運用**       | 外部CMS依存（XEE）以外は、固定的な画像配置が多く流動性が低い             |

---

## 3. 競合の弱点（Animoで超えるべき点）

1. **即時性の欠如** — 本日出勤機能がほぼ存在しない
2. **顔出し前提の設計** — 顔なし体験でのキャスト選択UIが未整備
3. **来店前体験の不足** — Night Style診断・ムード表示などが0
4. **予約フォームの不備** — 電話のみ多数 → 機会損失
5. **料金の透明性不足** — 初来店ハードルが高い
6. **スマホ体験の未整備** — 30秒予約完了設計の実装なし
7. **お気に入り機能なし** — リピーター向けの端末保存が存在しない
8. **没入感の欠如** — シャンデリア等の「店の売り」をアニメーションで体験化できていない
9. **求人導線の弱さ** — 顧客向けと求人向けの導線が混在、または内容が薄く応募時の不安を払拭できていない

## 4. Animoで採用する要素

| 要素                   | 参照元      | 採用理由                        |
| ---------------------- | ----------- | ------------------------------- |
| フロア別写真案内       | チック      | 来店ハードル低下                |
| FAQ設計（丁寧・具体）  | チック      | 初来店の不安解消に直結          |
| ドレスコード明記       | チック      | 高級感・客層フィルタリング      |
| 英語名の併記           | チック      | 国際感・接待ニーズへの対応      |
| キャスト個別URL        | XEE・チック | SEO + 指名予約の動機形成        |
| ニュース・イベント更新 | XEE         | 常時更新でSEOとリピーターを両立 |
| 新人キャスト特集       | プリンス    | 初来店動機・新鮮さの演出        |

---

## 5. Animoで避ける要素

| 要素                        | 理由                         |
| --------------------------- | ---------------------------- |
| 外部CMS依存（ファンクラ等） | ブランド世界観の棄損         |
| ポータルサイト的汎用UI      | 高級感との矛盾               |
| 顔出し前提のキャストカード  | 運用制約に反する             |
| 料金の完全非公開            | 初来店ハードルが過剰に上がる |
| ランキング投票・ゲームUI    | 高級ブランドとのトーン不一致 |

## 6. Animoの差別化案（総括）

| 項目                               | 差別化内容                                                    |
| ---------------------------------- | ------------------------------------------------------------- |
| 本日出勤リアルタイム表示           | 当日来店の即決トリガー（業界最高水準）                        |
| Night Style診断                    | 顔出し不可でもキャスト選択体験を成立させる                    |
| 今日の店内ムード                   | 来店前の期待値調整 → ミスマッチ防止                           |
| お気に入りキャスト（LocalStorage） | プライバシー配慮しつつリピーター体験を向上                    |
| ケース別料金シミュレーター         | 初来店の料金不安をUIで解消                                    |
| シャンデリア主役のビジュアル設計   | 店内の最大の強みをFVで象徴化                                  |
| 30秒予約導線                       | スマホで完結する予約UX（業界初水準）                          |
| VIP席空き状況                      | 接待層の予約判断を即時サポート                                |
| **シネマティック・アニメーション** | GPU最適化された高品質なスクロール連動演出で他店を圧倒         |
| **独立した最強の求人ページ**       | キャスト/スタッフ専用の待遇・不安払拭コンテンツと専用フォーム |
| **流動的画像プレースホルダ**       | 画像がなくても成立するUI骨組みと、運用負荷のない更新システム  |

# Club Animo Webサイト 管理画面 UX 詳細設計書

0 目的
本設計書はClub Animo Webサイトの管理画面UXを定義する。
現場のITリテラシーを考慮し、以下を重視する: 直感操作 / スマホ操作 / 更新の速さ / 誤操作防止。

---

1 管理画面URL / 2 ログイン画面

- URL: `/admin`
- 要素: ロゴ / メール入力 / パスワード入力 / ログインボタン / パスワード忘れ

---

3 管理画面レイアウト

- 左サイドバー（ダッシュボード, キャスト管理, 出勤管理, イベント管理, ニュース管理, ギャラリー管理, 求人応募管理, 問い合わせ管理）

---

4 ダッシュボード

- 表示内容: 本日の出勤人数 / キャスト総数 / 新着応募 / 新着問い合わせ

---

5 キャスト管理

- キャスト一覧: 写真 / 源氏名 / 年齢 / 表示状態 / 編集ボタン
- キャスト登録: 写真アップロード / 名前 / 年齢 / 身長 / 趣味 / コメント
- タグ設定: （例）清楚 / お姉さん / 可愛い

---

6 出勤管理

- 本日の出勤設定: 日付選択 / キャストチェック（チェックボックス方式） / 保存ボタン

---

7 イベント管理 / 8 ニュース管理

- イベント: 一覧 / 作成（タイトル, 開催日, 説明, 画像）
- ニュース: 一覧 / 作成（タイトル, 本文, 画像, 公開日時）

---

9 ギャラリー管理

- 店内写真管理: 画像アップロード / 説明 / 並び順

---

10 求人応募管理

- 応募一覧表示: 名前 / 年齢 / 電話番号 / 応募種別 / 日時
- ステータス: 未対応 / 連絡済 / 面接予定 / 不採用

---

11 問い合わせ管理

- 一覧表示: 名前 / 電話番号 / 問い合わせ内容 / 日時

---

12 検索機能 / 13 ソート機能

- キャスト検索 / 応募検索
- ソート: 日付順 / 名前順

---

14 通知 / 15 モバイルUI

- 新規応募通知 / 新規問い合わせ通知
- スマホから操作可能 / ボタン大きめ / 入力しやすいフォーム

---

16 操作ログ / 17 エラー表示 / 18 セキュリティ

- ログ: 更新履歴 / 作成者 / 更新日時
- エラー: 入力エラー / 保存失敗の表示
- セキュリティ: ログイン必須 / 自動ログアウト

---

19 将来拡張 / 20 受け入れ基準

- 将来: LINE通知 / 求人自動返信
- 受け入れ: 店舗スタッフが操作可能 / 出勤更新が1分以内 / キャスト登録簡単 / 応募確認可能

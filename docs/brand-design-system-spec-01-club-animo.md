# Club Animo Webサイト ブランドデザインシステム設計書

0 目的
本設計書はClub Animo Webサイトのブランドデザインシステムを定義する。
サイト全体のデザイン品質を統一し、拡張性・再利用性を高める。

---

1 ブランドコンセプト

- 明るい高級クラブ / 洗練された大人の空間 / 上品で落ち着いた雰囲気

---

2 カラーデザイン

- ベースカラー: 白 (RGB: 255 255 255) - 背景
- テキストカラー: ダークグレー (RGB: 17 17 17) - 本文
- アクセントカラー: ゴールド (RGB: 201 168 106) - CTA, ライン
- 補助カラー: ライトグレー (RGB: 245 245 245) - セクション背景

---

3 フォント設計

- 英字: 高級感のあるセリフ系 (Hero/タイトル)
- 日本語: 可読性の高いゴシック (本文)
- サイズ: Hero 48px / ページタイトル 36px / セクションタイトル 28px / 本文 16px

---

4 余白・カード・セクション設計

- セクション上下: 120px / コンテンツ間: 40px / カード内: 20px
- Primaryボタン: 背景ゴールド・文字白・角丸8px / Hover時拡大+影
- カードデザイン: 白背景・薄いシャドウ / Hover時軽く浮き上がり

---

8 〜 10 画像・アイコン・ナビゲーション

- 写真トーン: 暖色 / 明るめ / 高級
- アイコン: シンプルラインアイコン (黒・ゴールド)
- ナビゲーション: スクロール時にコンパクト表示 (白背景)
- CTA: ゴールドボタン (常時配置)

---

14 アニメーションスタイル

- 滑らか / 控えめ / 高級感 (フェードイン, スライドイン, ホバー)

---

20 受け入れ基準

- 全ページデザイン統一 / 高級感維持 / UIコンポーネント再利用可能 / ブランドイメージ保持

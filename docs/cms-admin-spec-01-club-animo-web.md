# Club Animo Webサイト CMS / 管理画面設計書

0. 目的
   本設計書はClub AnimoのWebサイトを店舗側で運用できるようにするためのCMSおよび管理画面仕様を定義する。
   キャバクラサイトは日々更新が必要な情報が多く、以下の更新を店舗スタッフ（専門知識を持たない担当者）が簡単に行えるようにする。

更新対象:

- 本日の出勤
- キャスト情報
- イベント
- ニュース
- 店内写真
- 求人情報

---

1 管理画面の基本方針

- シンプル
- 直感操作
- スマホ対応（スマホからも管理可能）
- 入力項目最小

---

2 ログイン権限管理

- URL: `/admin`
- 認証: メール / パスワード
- 権限: 管理者 / 店舗スタッフ

---

3 管理ダッシュボード
ログイン後のトップ表示内容:

- 本日の出勤人数
- キャスト人数
- イベント数
- 最新更新

---

4 キャスト管理
キャストの登録と編集を行う

- 源氏名
- 写真（顔出し不可前提、後ろ姿写真等）
- 年齢 / 身長 / 趣味 / 一言コメント
- タグ（例: 可愛い / 大人 / 清楚 / ギャル）
- 表示状態（公開 / 非公開）
- 並び順

---

5 キャスト写真管理

- 画像アップロード
- 画像比率: 4:5（UI定義に準拠）
- 自動リサイズ

---

6 本日の出勤管理

- 方法: キャストをチェック方式で出勤登録（一覧からの選択）
- 日付指定 / 出勤時間設定

---

7 所属キャスト一括表示

- キャスト全表示
- 並び替え: 人気順 / 新着 / 管理順

---

8 イベント管理

- タイトル / 開催日 / 説明 / 画像 / 公開状態

---

9 ニュース管理

- タイトル / 本文 / 画像 / 公開日時

---

10 ギャラリー管理（店内写真）

- 画像アップロード / 説明文 / 表示順

---

11 求人管理

- キャスト求人: 内容 / 給与 / 条件 / 応募方法
- スタッフ求人: 仕事内容 / 給与 / 応募方法

---

12 応募管理（求人フォーム）

- 応募一覧: 名前 / 年齢 / 連絡先 / 応募種別（キャスト・スタッフ） / メモ

---

13 お問い合わせ管理

- 問い合わせ一覧: 名前 / 連絡先 / 内容 / 返信状態

---

14 全体画像管理（プレースホルダ設計と連動）
サイト全体の画像を管理し、差し替え可能にする。

- Hero画像 / 店内写真 / キャスト写真

---

15 SNS・SEO・アクセス管理

- SNS: Instagramリンク（更新可能）
- SEO: ページごとの title / description
- アクセス: アクセス数表示

---

16 セキュリティ / 推奨技術

- ログイン制御 / 不正ログイン防止
- Stack: Next.js / Supabase / React

---

17 受け入れ基準

- 店舗スタッフが操作できる
- 出勤更新、キャスト更新が直感的で簡単
- 求人応募が管理できる
- スマホでも管理操作が可能

# Club Animo 実装・技術要件プロンプト

あなたは世界トップレベルのWEBエンジニア、WEBディレクター、UIデザイナーです。
以下の仕様書をすべて理解・分析したうえで、Club AnimoのWEBサイトを実装してください。
単なる店舗サイトではなく「高級ブランドサイトレベル」のUI/UXを実現してください。

参考サイト:

- https://www.club-xee.com/
- https://www.chick.co.jp/roppongi/
- https://princegroup.jp/
- https://www.clubepic.net/top/

重要条件:

- 背景は白 / 明るい高級感 / シャンデリアのある豪華な店内 / 箱が広い高級クラブ
- 客層は富裕層
- キャストは顔出し不可が多い (後ろ姿写真で掲載)
- モーション必須 (Heroフェード, スクロールアニメーション, Hoverエフェクト, カードアニメーション, ページトランジション)
- 推奨ライブラリ: Framer Motion または GSAP
- 技術スタック: Next.js, TypeScript, Tailwind CSS, Supabase, Vercel

実装内容:

- CMS管理画面 (`/admin`) 実装 (キャスト、出勤、イベント、ニュース、ギャラリー、求人応募、問い合わせ)
- プレースホルダベースの画像処理機能
- SEO、GEO最適化 (title, description, 構造化データ, FAQ)
- モバイル最適化、パフォーマンスLighthouse 90以上

必須ページ:

- トップ / キャスト一覧・詳細 / 本日の出勤 / 料金システム / ギャラリー / イベント / ニュース / キャスト求人 / スタッフ求人 / 問い合わせ
- 出勤機能は管理画面からチェック方式で更新可能にすること。

最後に以下を出力してください:
1 ディレクトリ構造
2 DBスキーマ
3 コンポーネント構造
4 ページ構成
5 API一覧

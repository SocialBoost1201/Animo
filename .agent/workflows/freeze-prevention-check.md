# Freeze Prevention Check

## Purpose
Antigravity の凍結やコンテキスト肥大を防ぐ。

## Steps
1. active skills 数を確認する
2. 今回のタスクに不要な skill を使わない
3. Shared Vault を live path として扱っていないか確認する
4. 大きな一覧を chat に貼らない
5. 詳細は Markdown に保存する
6. Antigravity が主担当である必要があるか確認する
7. 実装タスクなら Codex へ渡す
8. 深い監査なら Claude Code へ渡す

## Output
- active skill policy
- overflow risk
- recommended routing
